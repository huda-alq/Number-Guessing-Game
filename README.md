# Number Guessing Game

## Overview
This project is a browser based interactive number guessing game built using HTML, CSS, and JavaScript.  
The application generates a random number between 1 and 100, and the player attempts to guess it.  
The system provides real time feedback, tracks attempts, and maintains a simple leaderboard.

## Project Workflow
- Page initialization
- Player name input
- Random number generation
- Input validation
- Guess evaluation
- Attempt tracking
- Leaderboard update
- Game reset functionality
